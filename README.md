# farmer
##### 仿真果树化token应用
--------------------------

仿真果树token化研究应用软件是一个独立的基于区块链的供果农发布即将成熟收获的果树，供购买者竞价的web应用。

- 本系统是一个典型的Dapp应用，主要包含两部分，合约部分以及前端系统部分。
- 合约部分包含竞价信息合约，以及TRC树币合约。竞价信息合约会根据TRC树币合约发行的token数量，映射到相应数量的果树，发布相应数量的竞价信息列表。
- 前端部分通过以太坊公链与合约部分进行交互。完成相应的业务逻辑。
